# 💸 Budget Application

A comprehensive budget management web application that enables users to track income and expenses efficiently. Built with JavaScript, HTML, and CSS, the application offers an intuitive interface for adding, viewing, and managing financial transactions, with real-time updates and visual summaries.

## ✨ Features
- **Clock and Date Display**: Shows current time and date at the top for added context.
- **Add Transactions**: Easily input income and expense records with descriptions.
- **Category-Based Summaries**: Automatically categorize and display totals for income and expenses.
- **Detailed Financial Overview**: Displays total balance, percentage of expenses, and income vs. expense details.
- **Visual Feedback**: Interactive, color-coded display for transaction types, improving readability.

## 📖 Usage
- **Add Incomes and Expenses**: Input transactions directly to track your finances.
- **View Summary**: Check total income, expenses, and remaining balance, updated in real-time.
- **Remove Transactions**: Delete any income or expense to adjust the budget overview.
- **Responsive Clock**: Stay aware of the current date and time.

## 🛠️ Tech Stack
- **JavaScript** - Handles dynamic functionalities like data manipulation, real-time updates, and display toggling.
- **HTML** - Provides the application structure.
- **CSS** - Adds styling for an organized and visually appealing interface.
